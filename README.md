## 📋 项目概述

Vax Fridge 是一个创新的太阳能便携式超低温冷冻系统，专为确保mRNA疫苗从实验室到最偏远社区的"最后一公里"安全而设计。

## 🌟 主要特性

- **🌡️ 超低温控制**: -80°C超低温保存，确保mRNA疫苗稳定性和效力
- **☀️ 太阳能驱动**: 可持续清洁能源解决方案
- **📡 实时追踪**: GPS定位与云端数据同步
- **🛡️ GMP合规**: 符合国际医疗器械标准

## 🚀 技术栈

- **3D展示**: Three.js (备用模型优化)
- **样式**: 响应式设计，支持移动端
- **字体**: Google Fonts (Inter)
